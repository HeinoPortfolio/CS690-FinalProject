## Summary

This is repository the final project for Software Engineering. There was an assigned scenario.  A brief synopsis of the scenario is the following:

> Ava is seeking a way to manage her finances effectively.  Ava wants to save money by setting savings goals and tracking her daily expenses. The goal is to create a user-friendly application that will handle the day-to-day expenses and provide feedback on the progress toward her savings goals.
